# Simple OniReader

## Description

A lean library for parsing Primesense OpenNI ONI files.

## Maintainers

* Francisco Facioni <fran6co@gmail.com>

## Installation

```
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make install
```

## Testing

Using oni files from:
